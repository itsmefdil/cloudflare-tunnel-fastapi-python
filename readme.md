## Cloudflare Tunnel API with Python

### Prerequisites

- Python 3.10 or higher
- Cloudflare account
- Cloudflare API token

### Installation

```bash
pip install -r requirements.txt
```

### Usage

```bash
cp .env.example .env
```

```bash
uvicorn main:app --reload
```
